\section*{Inline and Paragraph Interfaces}

The command form is intended for inline use:
\bionic{Confucius said: Madam, I'm Adam.}
It does not force a paragraph break.

Different inline spans can use different fixation points:
\bionic[fixation-point=1]{internationalization}
and
\bionic[fixation-point=5]{internationalization}
are marked separately in the same paragraph.

\section*{Paragraph Environment}
